Formule 1

Een informatieve website waar je actuele Formule 1 informatie/statistieken kunt inzien van meerdere seizoenen - ondersteund door visuals.

  • Naar project
  • Code inzien

De Formule 1 data die ik voor deze informatieve website gebruik, is afkomstig van de Ergast API in de vorm van JSON. Deze API maakt het mogelijk om vanaf de eerste Formule 1 race (in 1950) tot de meest recente race de desbetreffende data op te halen. Dit is echter alleen data in tekstvorm en niet in de visuele vorm, waardoor ik zelf oplossingen moet bedenken over het tonen van foto's bij de relevante data.

De interface van de website heb ik zo minimalistisch mogelijkgehouden, zodat de content rustig/modern oogt en de ruimte krijgt. Het donkerblauw contrasteerd naar mijn mening mooi met het rood en de subtiele gradient laat het nog net iets moderner ogen. Verder is de stijl voornamelijk “flat” en maak ik gebruik van shadows waar nodig. Het gebruikte lettertype is ook het officiële lettertype van Formule 1. Aangezien dit hun eigen lettertype is, heb ik deze via de inspector van de officiële Formule 1 site moeten halen en importeren als een font in CSS. De content wordt voornamelijk getoond in een grid vorm, aangezien dit erg overzichtelijk is en makkelijk is aan te passen voor elk scherm. De website is dan ook volledig responsive, maar is voornamelijk gemaakt voor grotere schermen.
